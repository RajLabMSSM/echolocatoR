# Startup messages

Startup messages with info on package name, version, citation, GitHub,
etc.

## Usage

``` r
startup(
  package = "echolocatoR",
  add_batty = FALSE,
  color1 = cli::col_br_cyan,
  color2 = cli::col_br_magenta,
  color3 = cli::col_cyan
)
```

## Arguments

- package:

  R package name.

- add_batty:

  Add an ASCII bat.

- color1:

  First color to use in the palette.

- color2:

  Second color to use in the palette.

- color3:

  Second color to use in the palette.

## Value

Null
